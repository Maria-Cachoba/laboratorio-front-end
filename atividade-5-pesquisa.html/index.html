O que é a tag <strong>?

A tag <strong> no HTML é usada para destacar a importância de um trecho do texto. Ela normalmente deixa o texto em negrito, mas também informa ao navegador que aquele conteúdo tem valor semântico.

Exemplo de código da tag <strong>:

<!DOCTYPE html>




This text is normal.


É um texto importante!
